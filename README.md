# 🌿 SerenSpace

**SerenSpace** is an interactive, emotionally engaging mental wellness platform designed to provide instant relief, mindful guidance, and a safe space for self-reflection. It transforms the traditional static informational site into a living, breathing digital sanctuary.

---

## 🌟 Core Principles

- **Emotion-First UX**: Guided interactions designed to make the user feel supported, not overwhelmed.
- **Interactive Wellness**: Personalized suggestions based on how you feel *right now*.
- **Privacy & Trust**: Your data stays on your device. No clouds, no accounts, no tracking.
- **Micro-Interactions**: Smooth animations and tactile feedback using Framer Motion.
- **PWA Ready**: Installable on mobile and works offline for whenever you need it.

---

## 🎨 Design & Aesthetic

- **Visuals**: Modern, premium UI with soft pastel gradients (Purple → Blue → Green).
- **Glassmorphism**: Elegant card-based layouts with high-quality blur effects and subtle borders.
- **Typography**: Clean, readable sans-serif fonts (Inter & Lexend) designed for a calm reading experience.
- **Animations**: Fluid transitions and micro-animations that make the interface feel alive.

---

## 🛠️ Key Features

### 1. 🧠 Mood-Based Experience
- **"How are you feeling today?"**: A dynamic hero section where you select your mood via interactive emojis (🙂 😐 😞 😡 😰).
- **Personalized suggestions**:
  - **Anxious**: Breathing exercises & Grounding techniques.
  - **Sad**: Journaling prompts & Affirmations.
  - **Angry**: Physical release exercises & Muscle relaxation.
  - **Happy/Neutral**: Productivity tools & Focus modes.

### 2. ✍️ Smart Journal System
- **Guided Prompts**: No more blank-page anxiety. Rotating prompts like "What made you smile today?" guide your reflection.
- **Journal History**: Save your entries locally and look back at your journey through a clean, chronological log.

### 3. 🧘 Guided Breathing Exercise
- **Breathing Orb**: A beautiful, pulsing visual guide for "Inhale... Hold... Exhale" cycles.
- **Calming Visuals**: Soft gradients that expand and contract with your breath.

### 4. 📊 Progress Tracking
- **Check-in Streaks**: Celebrate your consistency with a daily streak counter.
- **Mood History**: Visualize your emotional patterns over time with a simple, intuitive history chart.

### 5. 🚨 Crisis Support
- **Immediate Help**: A sticky support section with direct access to helplines (India: 1800-599-0019).
- **Clear Disclaimer**: Professional resources are highlighted for urgent needs.

### 6. 📱 PWA (Progressive Web App)
- **Installable**: Add SerenSpace to your home screen on Android and iOS.
- **Offline Mode**: Essential assets are cached so you can find calm even without an internet connection.

---

## 🔒 Privacy & Data

- **LocalStorage Persistence**: All your journal entries, mood history, and streaks are stored securely in your browser's `localStorage`.
- **Zero Cloud**: We don't send your data to any server. What you feel and write is for your eyes only.

---

## 💻 Tech Stack

- **Framework**: [React 19](https://react.dev/) (with [Vite](https://vitejs.dev/))
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **PWA**: [vite-plugin-pwa](https://vite-pwa-org.netlify.app/)

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-repo/serenspace.git
   ```
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Run in development**:
   ```bash
   npm run dev
   ```
4. **Build for production**:
   ```bash
   npm run build
   ```
